## Development

### Setup

1. Clone the repository:
```bash
git clone https://github.com/HakumenNC/datashear.git
cd datashear
```

2. Install in development mode:
```bash
pip install -e ".[dev]"
```

### Testing

Run tests with:
```bash
pytest
```

### Building

Build the package:
```bash
python -m build
```
